redis-dump

Long & Kelly
2017-07-17
Redis
Redis
  • Github redis-dump

在尝试的过程中，有一个问题需要注意：

如果一个redis里面已经存在了某一个key，在redis-load的时候不能覆盖掉。

load的时候，不会删掉redis已经存在的数据。

测试简单的redis数据结构,key-value / hash set ／list 没有问题，建议使用的时候先测试下自己的数据结构能不能load成功回来。
