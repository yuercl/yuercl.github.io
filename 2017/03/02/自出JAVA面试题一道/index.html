自出JAVA面试题一道

Long & Kelly
2017-03-03
Java
Java

最近招人，想了几道面试题，包含基础知识、深入一点、Linux一点、数据库一点。比较基础，但是足够在简历筛选后再过滤一部分人了。其实面试题做的怎么样无所谓，聊一聊也大概知道是什么水平、能深入到什么水平。聊天的时候也还可以问问补充的题，比如一些框架基础、新技术趋势、实现原理什么的，另外可以考察下解决问题的思维方式和交流沟通怎么样，实在不行，还可以叫他现场写段代码。或许也就知道对方是什么水平了。感觉要遇到一个特别喜欢的求职者还是比较困难，当然也不只是技术。

接下来要做的事，就是实现之前想象的一个量化策略，时间不多了。

也祝我好运吧～，写在深圳，夜，BGM<刚好遇见你>，I count the days!!

  1. == 与 equals的区别
  2. 简要描述String、StringBuffer、StringBuilder的区别。
  3. Char型变量能否存储一个中文字？为什么？
  4. 请写出ArrayList、LinkedList和Vector的区别？
  5. 如何在两个线程间共享数据？
  6. 请写一个你最熟悉的设计模式的代码，一般在是什么应用场景？如：单例、工厂、观察者、代理、建造者、适配器等…
  7. java的内存回收机制是什么样的？java是否存在内存泄漏？有什么途径检测内存泄漏？（把知道的都可以写出来）
  8. 在Linux里，如何查看一个监听端口为10086的进程？如果要杀死它呢？
  9. 目前PP停车项目中有一张详细的出入场停车记录表 _record<long id, varchar16 merchant, datetime enter_time, datetime leave_time, varchar16 card_no> ，注: merchant 商户号, enter_time 入场时间, leave_time 出场时间。
    请写出SQL(数据库不限)来统计指定商户的停车场的今天里面的每小时车辆入场数量。
  10. 一条SQL执行过长的时间，你可以从哪些方面做优化？（希望写出思路）
  11. 以下代码片段存在的主要问题是什么？怎么解决？
  1. 下面这段代码输出结果是什么？为什么？
