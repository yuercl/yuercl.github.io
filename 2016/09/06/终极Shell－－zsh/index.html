如果你在用Shell，默认那用bash，那一定要试试zsh，配合oh-my-zsh那必须推荐，很骚，真的很骚～～

  • 官网地址: http://ohmyz.sh
  • Github: https://github.com/robbyrussell/oh-my-zsh

Your terminal never felt this good before.

一定要安装autojump神器.

使用 zsh

  • autojump j 这货支持模糊匹配，不用输全，一个字骚
  • kill 再按一个tab键，经常kill一个jar，效率很高
  • .. / … cd都不需要
  • tab 多按几下，有惊喜
  • git 一般大家都配置别名
  • 输入一个s，上下键自动提示之前匹配的输入的命令，比如直接出来ssh xxx,scp xxx
