使用 zsh

  • autojump j 这货支持模糊匹配，不用输全，一个字骚
  • kill 再按一个tab键，经常kill一个jar，效率很高
  • .. / … cd都不需要
  • tab 多按几下，有惊喜
  • git 一般大家都配置别名
  • 输入一个s，上下键自动提示之前匹配的输入的命令，比如直接出来ssh xxx,scp xxx
