Storm安装

2016年9月6日

前提需要一个zookeeper，一个节点或者一个cluster，接下来标准流程

  • wget …
  • tar -xzvf …
  • cd …
  • vim conf/storm.yaml

各种start以后

开了ui后会打开http的8080端口，里面能看到服务器的状态，之后提交一个Topology到Storm上，Topology 从提交的那一刻就会一直运行，接收spout的数据，直到手动结束
